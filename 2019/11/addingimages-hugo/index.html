Adding images on Hugo

-이미지 가운데 정렬로 .md문서에 넣기

  1. 이미지 노출할 때 사이즈와 가운데 정렬을 style로 지정하여 적용한다.
    1) (Local저장소) \blog\themes\hugo-tranquilpeak-theme\static\css 폴더에서 style.css 파일을 아래 경로에 복사한다.
    2) (Local 저장소) \blog\static\css\style.css 로 복사된 css 파일에 아래 style을 추가한다.

    img[src$='#center30'] {
display: block;
margin-left: auto;
margin-right: auto;
width: 30%;
}

img[src$='#center50'] {
display: block;
margin-left: auto;
margin-right: auto;
width: 50%;
}

img[src$='#center75'] {
display: block;
margin-left: auto;
margin-right: auto;
width: 75%;
}

img[src$='#center100'] {
display: block;
margin-left: auto;
margin-right: auto;
width: 100%;
}

  2. (Local 저장소) blog/layouts/partials/header.html 에 위 1번의 style.css 파일을 추가한다.

    <link rel="stylesheet" href="{{ "/css/styles.css" | relURL }}">

  3. .md 파일을 아래 정보를 참고하여 이미지를 추가한다.
    e.g. (Local저장소) blog\static\images\addingImages-hugo\lemmings_01.png

    image 저장 위치: (Local저장소) blog\static\images\
    md파일은 아래와 같이 작성할 때 추가하면 된다.

    //      이미지 파일 경로와 파일명     #css style
![](/images/blog_construct/html0.png#center75)

  4. hugo server -D 로 테스트 후 github push 하면 된다.
