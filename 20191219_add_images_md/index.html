md 문서에 이미지 넣기

December 19, 2019 in Blog w/Hugo
  • 내부 저장소와 외부 저장소의 이미지를 추가합니다. Update: Dec 19, 2019

1. md 파일에 아래 정보를 참고하여 이미지를 추가합니다.

  • {Blog}/static/_images 로 되어있다면 (관리상 동일하게) {Blog}/static/images 로 rename 후 테스트해보세요.

    e.g. (내부 저장소)

    저장 경로: {Blog}/static/images/20191219_add_images_md/lemmings_01.png
호출 경로: ![로컬 이미지](/images/20191219_add_images_md/lemmings_01.png)

    e.g. (외부 저장소)

    ![외부 이미지](https://s.yimg.com/rz/p/yahoo_homepage_en-US_s_f_p_bestfit_homepage_2x.png)

요약

![ {Alt text 생략가능} ](/images/관리 목적의 폴더명/이미지 파일.png)
![ {Alt text 생략가능} ](https:// ... 이미지 URL ... )

2. hugo server -D 로 테스트 후 git push.
